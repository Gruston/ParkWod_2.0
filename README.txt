# PARK WOD v7 - Install on Your Phone

## Quick Install (iPhone or Android)

### Option A: Host for Free on GitHub Pages (Recommended)
1. Go to https://github.com and create a free account (if you don't have one)
2. Click "New Repository" → name it "parkwod" → make it Public
3. Upload ALL files from this folder (index.html, manifest.json, sw.js, icon-192.png, icon-512.png)
4. Go to Settings → Pages → Source: "main" branch → Save
5. Wait 1-2 minutes, then visit: https://YOUR-USERNAME.github.io/parkwod/
6. On your phone, open that URL in Safari (iPhone) or Chrome (Android)
7. **iPhone**: Tap the Share button → "Add to Home Screen"
8. **Android**: Tap the 3-dot menu → "Install app" or "Add to Home Screen"

### Option B: Host on Netlify (Even Easier)
1. Go to https://app.netlify.com/drop
2. Drag the entire parkwod-app folder onto the page
3. It gives you a URL instantly
4. Open that URL on your phone
5. Add to Home Screen (same as above)

### Option C: Host on Vercel
1. Go to https://vercel.com and sign up free
2. Create new project, upload the files
3. Get your URL and add to Home Screen

## What You Get
- Works offline (no internet needed after first load)
- Full-screen app experience (no browser toolbar)
- 208 workouts with smart timers
- Workout history logging
- Exercise encyclopedia with 57 exercises
- 5-second countdown before timed workouts
- Filter workouts by equipment, difficulty, format

- index.html - The complete app
- manifest.json - Tells the phone this is an app
- sw.js - Service worker for offline support
- icon-192.png - App icon (small)
